# NLP-System-to-Identify-Mispelled-Words-and-use-NLP-Word-Prediction-Techniques-to-Correct-Them.

The aim of this assignment was to take in a series of sentences which have mispelled words and correct them. 

Some of the items utilized were:
- WordNet
- Bigrams
- Tri-grams
- Named Entity Recognition
- Add-one Smoothing
